# Privacy Policy — Probably AI

*Last updated: April 1, 2026*

## Summary

Probably AI does not collect, store, or transmit any user data.

## Details

- **No data collection:** This extension does not collect any personal information or browsing data.
- **No network requests:** This extension makes no network calls. All processing happens locally in your browser.
- **Local storage only:** User preferences (such as enabled/disabled state and detection rules) are stored using Chrome's built-in storage API and never leave your browser.
- **No third parties:** No data is shared with or sent to any third-party services.
- **No analytics:** This extension contains no analytics, tracking, or telemetry of any kind.

## Contact

If you have questions, please open an issue at https://github.com/taekim/probably-ai/issues.
